Análise da Evolução do Desenvolvimento Socioeconômico do Brasil em Diferentes Contextos Inflacionários

Author

Felipe Lobo

Introdução

Este trabalho tem como objetivo analisar a trajetória socioeconômica do Brasil ao longo da década de 2010 a 2020. Para isso, foram selecionados três indicadores fundamentais: o Produto Interno Bruto (PIB), que mensura a atividade econômica do país; o Índice de Desenvolvimento Humano (IDH), que reflete a qualidade de vida da população através de componentes de saúde, educação e renda; e o Índice Nacional de Preços ao Consumidor Amplo (IPCA), utilizado como medida da inflação oficial.

A análise se concentra no período entre 2010 e 2020 e busca investigar as correlações e tendências entre esses indicadores. Através da limpeza, integração e visualização dos dados obtidos do IPEADATA, o relatório procura responder à seguinte questão: como o crescimento econômico e o desenvolvimento social se relacionaram no Brasil durante uma década de variada instabilidade inflacionária?

Glossário de Variáveis

Variável Descrição
ano Ano da observação (2010 a 2020).
pib_total_anual Soma do PIB mensal de fontes do IPEADATA, representando o PIB total do ano.
idh Valor do Índice de Desenvolvimento Humano para o ano.
ipca_medio_anual Média da variação mensal do IPCA para o ano.

df_idh_raw <- read_excel("H:\\Documentos\\FELIPE\\R\\ipeadata[04-10-2025-06-59].xls")

# Limpando o IPCA - mensal para anual
df_ipca_anual <- df_ipca_raw %>%
  rename(ano_mes = Data, ipca_variacao = `IPCA`) %>%
  mutate(ano = floor(ano_mes)) %>%
  filter(ano >= 2010 & ano <= 2020) %>%
  group_by(ano) %>%
  summarise(ipca_medio_anual = mean(ipca_variacao, na.rm = TRUE))

# Limpando o PIB - mensal para anual
df_pib_anual <- df_pib_raw %>%
  rename(ano_mes = Data, pib_valor_mensal = `PIB - R$ (milhões)`) %>%
  mutate(ano_mes = as.numeric(ano_mes)) %>%
  mutate(ano = floor(ano_mes)) %>%
  filter(ano >= 2010 & ano <= 2020) %>%
  group_by(ano) %>%
  summarise(pib_total_anual = sum(pib_valor_mensal, na.rm = TRUE))

# Pivotando o IDH
df_idh_filtrado <- df_idh_raw %>%
  pivot_longer(cols = matches("^20\\d{2}$"), names_to = "ano", values_to = "idh") %>%
  mutate(ano = as.numeric(ano), idh = as.numeric(str_replace(idh, ",", "."))) %>%
  select(ano, idh)

# Integração das bases de dados
df_final <- left_join(df_pib_anual, df_idh_filtrado, by = "ano")
df_final <- left_join(df_final, df_ipca_anual, by = "ano")

Interpretação e Conclusão

É possível ver na matriz de correlação uma forte correlação entre PIB e IDH, e o gráfico “Evolução do IDH e do PIB Total Anual (2010–2020)” mostra isso de forma clara. Já o IPCA tem correlação menor e negativa com o IDH, mostrando que menos inflação pode sim ser bom para o IDH; mas, se o produto do país estiver aumentando, o IDH tende a subir.
